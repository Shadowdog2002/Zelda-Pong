# Pong

A two-player Pong game built with Pygame, featuring Link and Zelda as the paddles and item pickups on the stage.

Play it in the browser: [GitHub Pages](https://shadowdog2002.github.io/Zelda-Pong/)

## Controls

### Player 1 (Left)
| Key | Action |
|-----|--------|
| W | Move Up |
| S | Move Down |

### Player 2 (Right)
| Key | Action |
|-----|--------|
| Up Arrow | Move Up |
| Down Arrow | Move Down |

### General
| Key | Action |
|-----|--------|
| R | Reset game |
| Space | Pause / Unpause |

## Debug Controls

| Key | Action |
|-----|--------|
| C | Toggle debug mode |
| M | Spawn an extra ball |
| L | Rotate ball velocity clockwise |
| K | Rotate ball velocity counter-clockwise |
| Right Arrow *(while paused)* | Step forward one frame |
| Mouse Wheel | Increase / decrease FPS |

## Running Locally

```bash
pip install pygame pygbag
python main.py
```
